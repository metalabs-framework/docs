# Events

meta_core fires server-side events throughout the player lifecycle. All events are non-networked (`AddEventHandler`, not `RegisterNetEvent`).

## Core Ready

Fired once when meta_core has finished initializing.

```lua
AddEventHandler('meta_core:server:ready', function()
    print('meta_core is ready')
end)
```

## Player Lifecycle

### playerLoaded

Fired when a player's character is set for the first time. This is the canonical "player fully loaded" event.

```lua
AddEventHandler('meta_core:server:playerLoaded', function(source, acct, character)
    print(character.fullname, 'has loaded')
end)
```

### playerUnloading

Fired when a loaded player is being disconnected (before cleanup).

```lua
AddEventHandler('meta_core:server:playerUnloading', function(source)
    print('Player', source, 'is unloading')
end)
```

### playerConnected

Fired immediately after a new player connection is created.

```lua
AddEventHandler('meta_core:server:playerConnected', function(source, acct)
    print(acct.name, 'connected')
end)
```

### playerDisconnecting

Fired before a player connection is removed. Data is still accessible at this point.

```lua
AddEventHandler('meta_core:server:playerDisconnecting', function(source)
    -- Last chance to read player data
end)
```

### playerDisconnected

Fired after a player connection has been fully removed.

```lua
AddEventHandler('meta_core:server:playerDisconnected', function(source, identifier)
    print('Player', source, 'disconnected')
end)
```

## Character Events

### charCreated

Fired when a new character is created via `PlayerCharCreate`.

```lua
AddEventHandler('meta_core:server:charCreated', function(source, character)
    print(character.fullname, 'created in slot', character.slot)
end)
```

### charDeleted

Fired when a character is deleted via `PlayerCharDelete`.

```lua
AddEventHandler('meta_core:server:charDeleted', function(source, charId)
    print('Character', charId, 'deleted by', source)
end)
```

### charSelected

Fired when a character is selected via `PlayerCharSelect`.

```lua
AddEventHandler('meta_core:server:charSelected', function(source, character)
    print(source, 'selected', character.fullname)
end)
```

### charLoaded

Fired when a character is set as active.

```lua
AddEventHandler('meta_core:server:charLoaded', function(source, character)
    print(character.fullname, 'loaded')
end)
```

### charUnloaded

Fired when an active character is cleared.

```lua
AddEventHandler('meta_core:server:charUnloaded', function(source)
    print('Character unloaded for', source)
end)
```

## Job & Gang Events

### jobUpdated

Fired when a player's job is changed via `PlayerSetChar(source, 'job', ...)`.

```lua
AddEventHandler('meta_core:server:jobUpdated', function(source, job, oldJob)
    print('Job changed to', job.name, 'grade', job.grade)
end)
```

### gangUpdated

Fired when a player's gang is changed via `PlayerSetChar(source, 'gang', ...)`.

```lua
AddEventHandler('meta_core:server:gangUpdated', function(source, gang, oldGang)
    print('Gang changed to', gang.name)
end)
```

## State Events

### stateChanged

Fired when a single state value is changed via `PlayerSetState`.

```lua
AddEventHandler('meta_core:server:stateChanged', function(source, key, value, oldValue)
    print(key, 'changed from', oldValue, 'to', value)
end)
```

### stateBatchChanged

Fired when multiple state values are changed via `PlayerSetStates`.

```lua
AddEventHandler('meta_core:server:stateBatchChanged', function(source, changes)
    for key, entry in pairs(changes) do
        print(key, entry.old, '->', entry.new)
    end
end)
```
