# Permissions

Permissions in meta_core are built on FiveM's ACE system. Admin levels are determined by ACE principals assigned to a player.

## Admin Levels

| Level | Value | ACE |
|-------|-------|-----|
| NONE | 0 | — |
| MODERATOR | 1 | `meta.moderator` |
| ADMIN | 2 | `meta.admin` |
| SUPERADMIN | 3 | `meta.superadmin` |
| GOD | 4 | `meta.god` |

Higher ACE levels include all lower levels. A player with `meta.admin` has admin level 2.

## Setting Up ACE Permissions

In your `server.cfg`:

```cfg
add_ace group.moderator meta.moderator allow
add_ace group.admin meta.admin allow
add_ace group.superadmin meta.superadmin allow
add_ace group.god meta.god allow

add_principal identifier.fivem:12345 group.admin
```

## Exports

### PlayerHasPermission

Check if a player has a specific ACE permission node.

```lua
local allowed = exports.meta_core:PlayerHasPermission(source, 'meta.admin')
```

### PlayerHasRole

Check if a player has a specific role.

```lua
local isMod = exports.meta_core:PlayerHasRole(source, 'meta.moderator')
```

### PlayerHasJob

Check if a player has a specific job, optionally with a minimum grade.

```lua
local isPolice = exports.meta_core:PlayerHasJob(source, 'police')
local isSgt = exports.meta_core:PlayerHasJob(source, 'police', 3)
```

### PlayerHasGang

Check if a player is in a specific gang, optionally with a minimum grade.

```lua
local isBalla = exports.meta_core:PlayerHasGang(source, 'ballas')
local isOG = exports.meta_core:PlayerHasGang(source, 'ballas', 3)
```

### PlayerIsAdmin

Check if a player meets a minimum admin level. Defaults to `MODERATOR` (1) if no level specified.

```lua
local isAdmin = exports.meta_core:PlayerIsAdmin(source)
local isSuperAdmin = exports.meta_core:PlayerIsAdmin(source, 3)
```

### PlayerAdminLevel

Get the player's admin level as a number.

```lua
local level = exports.meta_core:PlayerAdminLevel(source)
```

## Using meta.player directly

```lua
meta.player.hasPermission(source, 'meta.admin')
meta.player.hasJob(source, 'police', 3)
meta.player.isAdmin(source)
meta.player.adminLevel(source)
```
