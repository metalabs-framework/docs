# Formatting

The format module provides currency, number, time, and string formatting utilities. Available on both client and server.

## Currency

#### FormatCurrency

```lua
exports.meta_lib:FormatCurrency(50000)       --> "$50,000"
exports.meta_lib:FormatCurrency(1234.56)     --> "$1,234.56"
exports.meta_lib:FormatCurrency(-500)        --> "-$500"
exports.meta_lib:FormatCurrency(1000, false) --> "$1,000" (hide cents)
```

#### Compact

Abbreviate large numbers:

```lua
lib.format.Compact(1500)       --> "1.5K"
lib.format.Compact(2500000)    --> "2.5M"
lib.format.Compact(1000000000) --> "1B"
```

#### Percent

```lua
lib.format.Percent(0.075)    --> "7.5%"
lib.format.Percent(0.5, 0)  --> "50%"
```

## Numbers

#### FormatNumber

```lua
exports.meta_lib:FormatNumber(1234567)     --> "1,234,567"
exports.meta_lib:FormatNumber(1234.5, 2)   --> "1,234.50"
```

## Time

#### FormatTime

```lua
exports.meta_lib:FormatTime(90)             --> "1m 30s"
exports.meta_lib:FormatTime(3661)           --> "1h 1m"
exports.meta_lib:FormatTime(90061)          --> "1d 1h"
exports.meta_lib:FormatTime(90, 'clock')    --> "1:30"
exports.meta_lib:FormatTime(3661, 'clock')  --> "1:01:01"
```

#### TimeAgo

```lua
lib.format.TimeAgo(os.time() - 120)     --> "2m ago"
lib.format.TimeAgo(os.time() - 7200)    --> "2h ago"
lib.format.TimeAgo(os.time() - 172800)  --> "2d ago"
```

#### Timestamp

```lua
lib.format.Timestamp()                  --> "2025-03-18 14:30:00"
lib.format.Timestamp(os.time() - 3600)  --> "2025-03-18 13:30:00"
```

## Strings

```lua
lib.format.Trim('  hello  ')              --> "hello"
lib.format.Capitalize('hello world')      --> "Hello world"
lib.format.TitleCase('hello world')       --> "Hello World"
lib.format.Truncate('long text here', 8)  --> "long ..."
lib.format.PadLeft('42', 5, '0')          --> "00042"
lib.format.PadRight('hi', 5)              --> "hi   "
lib.format.Split('a, b, c', ',')          --> { "a", "b", "c" }
lib.format.Slugify('Hello World!')        --> "hello-world"
```
