# Transactions & Ledger

Every balance mutation in meta_eco writes an immutable transaction record. This provides a complete audit trail for all financial activity.

## Transaction Record

Each transaction stores:

| Field | Description |
|-------|-------------|
| `account_id` | The account that was affected |
| `transaction_type` | `deposit`, `withdraw`, `transfer`, `invoice`, `payroll`, `tax`, `refund`, `system` |
| `amount` | The amount involved |
| `balance_before` | Balance before the mutation |
| `balance_after` | Balance after the mutation |
| `reason` | Auditable reason code (e.g. `salary`, `purchase`, `loan_repayment`) |
| `description` | Optional human-readable description |
| `related_account_id` | The other account in a transfer |
| `related_transaction_id` | Linked transaction (e.g. debit side of a transfer) |
| `source_resource` | The resource that initiated the mutation |
| `source_player` | The player who triggered it |
| `metadata` | Optional JSON metadata |
| `created_at` | Timestamp |

## Transfers Create Pairs

A transfer writes **two** linked transactions — a debit on the source account and a credit on the destination. They reference each other via `related_transaction_id`.

## Query Exports

#### GetTransactionsByAccount

Get transactions for a specific account, newest first.

```lua
local txns = exports.meta_eco:GetTransactionsByAccount(accountId, 50, 0)
for _, tx in ipairs(txns) do
    print(tx.transaction_type, tx.amount, tx.reason, tx.created_at)
end
```

#### GetTransactions

Query transactions with flexible filters.

```lua
local txns = exports.meta_eco:GetTransactions({
    account_id = accountId,
    transaction_type = 'transfer',
    reason = 'bank_transfer',
    source_player = charId,
    from_date = '2025-01-01',
    to_date = '2025-12-31',
    limit = 100,
    offset = 0,
})
```

## Ledger Exports

The ledger module wraps transactions with higher-level aggregation.

#### GetLedger

Get ledger entries for an account with optional filters.

```lua
local entries = exports.meta_eco:GetLedger(accountId, {
    transaction_type = 'deposit',
    limit = 20,
})
```

#### GetLedgerSummary

Get aggregated financial summary for an account over a time period.

```lua
local summary = exports.meta_eco:GetLedgerSummary(accountId, 30)
-- returns:
-- {

-- }
```
